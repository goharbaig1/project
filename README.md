# 📈 Stock Market Forecasting Using Machine Learning and Ensemble Learning

## Overview

This project was developed as part of my MSc Data Science dissertation at the University of Hertfordshire.

The objective was to investigate the effectiveness of machine learning and deep learning models for stock price forecasting using historical London Stock Exchange (LSE) data.

The study evaluates Long Short-Term Memory (LSTM), Random Forest (RF), and Gradient Boosting Machines (GBM), while exploring ensemble learning techniques to improve forecasting robustness and model generalisation.

---

## Research Objective

**Research Question:**

> Can machine learning models such as LSTM, Random Forest, and Gradient Boosting Machines accurately forecast stock prices, and how effectively do these models generalise to unseen data and future market conditions?

## Methodology

### Data Preprocessing

The following preprocessing techniques were applied:

- Missing value handling
- Exploratory Data Analysis (EDA)
- RobustScaler normalization
- Time-series windowing
- Feature preparation for machine learning models

### Models Developed

Three independent forecasting approaches were implemented:

1. Long Short-Term Memory (LSTM)
2. Random Forest Regressor (RF)
3. Gradient Boosting Machines (GBM)

Each model was trained separately on all five training stocks.

**Total Models Trained:**

```text
5 Stocks × 3 Algorithms = 15 Predictive Models
```

---

## Ensemble Learning Strategy

No single model consistently outperformed the others across all stocks.

To address this limitation, ensemble forecasting techniques were implemented.

### Stage 1: Model Training

For each stock, separate models were trained using:

- LSTM
- Random Forest
- Gradient Boosting Machine

### Stage 2: Ensemble Forecasting

Predictions generated by all trained models of the same algorithm type were combined through averaging:

- Ensemble LSTM
- Ensemble Random Forest
- Ensemble GBM

### Stage 3: Ensemble Selection

The final forecasting approach was selected based on:

- Mean Squared Error (MSE)
- R² Score

The ensemble producing the most favourable evaluation metrics was used for future stock price forecasting.

This strategy improved forecasting stability and reduced dependence on any individual model.

---

## Model Evaluation

Performance was evaluated using:

- Mean Squared Error (MSE)
- R² Score

Evaluation was performed on:

- Historical unseen stock data
- Future forecasting scenarios

---

## Key Findings

- LSTM models effectively captured long-term temporal dependencies.
- Random Forest and GBM performed strongly under specific market conditions.
- No single algorithm consistently outperformed all others.
- Ensemble learning produced more robust and stable forecasts.
- Forecast accuracy improved when new market data was periodically incorporated.
- Long-term stock price forecasting remained challenging due to market volatility and external economic factors.
